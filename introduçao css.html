introduçao css

Basicamente, o CSS permite que os desenvolvedores de páginas da web alterem o estilo, o tamanho, a cor, a fonte e outros aspectos visuais de elementos HTML. Por exemplo, é possível definir a cor de fundo de uma página, o tamanho e a fonte do texto, a cor e o estilo da borda de um elemento, etc..
O CSS foi criado para separar a apresentação do conteúdo da estrutura do documento, permitindo que o mesmo documento HTML possa ter diferentes estilos, sem precisar alterar o conteúdo. Assim, o desenvolvedor pode criar uma única folha de estilo que define as regras de estilo para todas as páginas da web de um site, tornando mais fácil e rápido fazer mudanças de design em todo o site.
